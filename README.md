# IRremoteControlledCar
This project demonstrates the design and implementation of an IR (Infrared) Remote Controlled Car using a microcontroller. The car responds to IR remote signals to perform basic movements such as forward, backward, left, right, and stop.

# Hardware

Microcontroller (Arduino Uno), 
IR Receiver Module (e.g., VS1838B), 
IR Remote, 
Motor Driver  (L293D), 
DC Motors 2, 
Wheels, 
Battery Pack, 
Jumper Wires, 
Breadboard

# Software Requirements

Arduino IDE, 
IRremote Library, 
USB cable for programming
